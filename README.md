CodeAlpha Language Translator

A web-based Language Translation Application developed using Python and Flask. This project enables users to convert text from one language into another through a clean and interactive interface.

✨ Key Features

- Translate text across various languages
- Easy-to-use and responsive user interface
- Instant translation results
- Supports multiple language selections

 🔧 Tech Stack

- Python
- Flask Framework
- HTML
- CSS

 ▶️ Getting Started

 Step 1: Install Dependencies
pip install flask

Step 2: Launch the Application
python app.py

 Step 3: Open in Browser
http://127.0.0.1:5000

📖 Sample Translation

Input: Good Morning

Translated Output: Bonjour (French)
